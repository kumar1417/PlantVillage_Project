# PlantVillage Project

## Course

ISB46703 – Principle of Artificial Intelligence

## Domain

Agriculture Health

## Dataset

PlantVillage Dataset (5 Classes)

## Models Used

* ResNet50
* DenseNet121
* MobileNetV3

## Evaluation Metrics

* Accuracy
* Mean Average Precision (mAP)
* Training Time

## Project Structure

* Data Preparation
* Data Modelling
* Data Visualization

## Conclusion

The three CNN models were evaluated for plant disease classification. MobileNetV3 achieved the best balance between classification accuracy and training efficiency, making it the most suitable model for this task.
